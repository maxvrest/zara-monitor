## Key Files Explained

### Extension Files
- **manifest.json**: Tells Chrome what the extension does and what permissions it needs
- **content.js**: Runs on Zara product pages, adds the monitor button, extracts product info
- **popup.html/js**: The settings interface when you click the extension icon
- **styles.css**: Makes the monitor button look nice

### Monitoring System
- **monitor.py**: The script that runs daily to check prices
- **monitor.yml**: Tells GitHub Actions when and how to run the monitoring
- **monitored_products.json**: Stores all products you're tracking with their prices

### Setup & Testing
- **test_telegram.py**: Run this to verify your Telegram bot works
- **requirements.txt**: Lists Python packages needed for the monitoring script

## What You Need to Edit
1. **Nothing in the extension folder** - it works as-is
2. **GitHub Secrets** - add your TELEGRAM_BOT_TOKEN
3. **Extension settings** - add your Telegram Chat ID

## What Gets Modified Automatically
- **monitored_products.json** - updated when you add/remove products and when prices are checked
